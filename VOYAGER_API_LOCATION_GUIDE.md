# Finding Existing Voyager API Settings

Since Fleetio already had this working, the API is definitely enabled. Here's where to look:

## Most Likely Locations:

### 1. **Reports Section**
- Click **"Reports"** in the left sidebar
- Look for:
  - **"Data Export"**
  - **"API Access"**
  - **"Integrations"**
  - **"Third-Party Integrations"**
  - **"Fleetio Integration"** (might still be listed)

### 2. **Organization Settings**
- Click **"Organization settings"** in the left sidebar
- Check ALL tabs/sections:
  - **"Integrations"** tab
  - **"API Settings"** tab
  - **"Data Access"** tab
  - **"Third-Party Apps"** tab
  - **"Connected Apps"** tab

### 3. **Transactions Section**
- Click **"Transactions"** in the left sidebar
- Look for:
  - **"Export"** button (might have API options)
  - **"Data Integration"** link
  - **"API Access"** link

### 4. **Cards Section**
- Click **"Cards"** in the left sidebar
- Check for:
  - **"Settings"** or gear icon
  - **"Data Export"**
  - **"Integration Settings"**

### 5. **Account Profile/Settings** (Top Right)
- Click your name/profile icon (top right)
- Look for:
  - **"API Settings"**
  - **"Developer Access"**
  - **"Integration Settings"**

## What to Look For:

- **API Key** or **Access Token**
- **API Endpoint** URL
- **Webhook URL** (if Fleetio had webhooks set up)
- **Integration Settings** page
- **Connected Apps** list (Fleetio might still be listed)
- **Data Export** settings

## Quick Check:

1. Look for any **gear icon** ⚙️ or **settings** links
2. Check for **"Export"** buttons - sometimes API access is under export options
3. Look for **"Fleetio"** anywhere - if it's still connected, you can see how it's configured
4. Check the **bottom of any page** - sometimes API links are in the footer

## If You Find Fleetio Still Connected:

- Click on it to see:
  - API endpoint they're using
  - How authentication is set up
  - What data is being synced

This will tell us exactly what we need to replicate for FleetPulse!
