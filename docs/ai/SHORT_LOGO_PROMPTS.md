# Short Logo Prompts (GPT-4o Character Limit)

## 🎯 Ultra-Short Version (Fits GPT-4o Limit)

**Use this concise prompt:**

```
Fleet management logo: Commercial truck silhouette (NOT minivan) with bold pulse line extending left, forming sharp peaks. Gradient indigo blue to purple. Text: "FleetPulse" bold sans-serif. Professional, modern, tech aesthetic.
```

## 📏 Slightly Longer (If Space Allows)

```
Professional fleet logo: Stylized commercial truck/delivery van outline (NOT minivan/SUV). Bold pulse/heartbeat line extends left from vehicle, creating sharp "M" peaks. Gradient: indigo blue (#6366f1) to purple (#9333ea). Text below: "FleetPulse" in bold modern sans-serif, dark indigo. Clean, minimalist, tech-forward design.
```

## 🔥 Most Important Elements (Priority Order)

If you need to cut even more, keep these essentials:

1. **"Commercial truck NOT minivan"** - Critical!
2. **"Pulse line extends left"** - Prevents cramped look
3. **"Gradient indigo to purple"** - Brand colors
4. **"FleetPulse text"** - Brand name

## 💡 Alternative: Split Into Two Prompts

**Prompt 1 (Icon only):**
```
Fleet management icon: Commercial truck silhouette (NOT minivan) with bold pulse line extending left, sharp peaks. Gradient indigo blue to purple. Minimalist, professional.
```

**Prompt 2 (Add text separately):**
```
Add "FleetPulse" text below icon, bold modern sans-serif, dark indigo color.
```
