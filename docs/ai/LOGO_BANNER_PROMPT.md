# FleetPulse Banner Logo Prompt for Bing Image AI

## Banner-Style Logo Prompt

**Prompt:**
```
Wide horizontal banner logo for fleet management software. Commercial delivery truck silhouette (box truck or van, NOT minivan) on left side. Bold pulse/heartbeat line extends from truck forming sharp peaks creating "M" or "W" shape. Text "FLEETPULSE" in bold sans-serif capital letters on right side. Gradient indigo blue (#6366f1) to purple (#9333ea). White text. Professional, modern, tech aesthetic. Horizontal layout, 1200x300 pixels, transparent background.
```

## Alternative Shorter Prompt (if character limit)

**Prompt:**

Wide banner logo: Commercial truck silhouette left, pulse line with peaks, "FLEETPULSE" text right. Indigo to purple gradient. White text. 1200x300px, transparent background.

## Specifications

- **Dimensions:** 1200px width × 300px height (4:1 aspect ratio)
- **Format:** PNG with transparent background
- **Style:** Horizontal banner layout
- **Colors:** Indigo (#6366f1) to Purple (#9333ea) gradient
- **Text:** "FLEETPULSE" in bold, white, sans-serif capitals
- **Icon:** Commercial truck/van silhouette with pulse line
- **Use Case:** Hero section logo placement

## Notes

- The banner format will work better for the hero section than a square logo
- Make sure the text is large and readable
- The pulse line should be prominent and dynamic
- Keep it professional and modern
